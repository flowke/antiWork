创意如虎添翼
负责处理，游刃有余。

伴随艺术家友好的3D绘图工具，你需要充分利用你的创造力，结合强大的工具来处理最复杂的资源， MARI是当今极具挑战性的视觉特效，动画和游戏制作的明智选择。支持了世界上最先进的分层系统， MARI能够处理高分辨率纹理与数以百万计的多边形而不牺牲速度和效率 。它还提供了一个完全可定制的用户界面，可以让你的工作只是你想要的方式。

能够处理数百万多边形和多个高分辨率的纹理无明显放缓， 让MARI负责的技术问题，你可以集中你精力在更重要的细节处理上。强大的，直观的分层系统可以让你用流畅和灵活的方式把procedurals ，滤镜，色彩校正和遮罩的混合效果直接画到3D模型上。

拥有直接支持阿诺德的V-Ray, Redshift 和 Unreal着色器，以及带MODO完全集成的工作流渲染和烘焙实时交互环境， MARI会让你有信心看到你希望的纹理表现。正如你对The Foundry期待的那样， MARI提供各种定制，让它像你希望的那样配合你。

为什么是 MARI?

  • 快速，直观绘制在3D模型上：一个与艺术家友好的绘图工具强大的分层，调整， procedurals和先进的遮罩。
  • 保持你的创造力澎湃不绝：即使是最复杂的资源和分辨率最高的纹理也不会让你跳出思考环境。
  • 工作在高保真的环境：基于物理的BRDF阴影和HDR环境光的工作状态让贴图更接近在目标平台的表现。
  • 消除令人沮丧且耗时多应用工作流：你可能烦恼过集成在阿诺德的V-Ray ，Redshift 和 Unreal着色器以及MODO渲染和烘培的工作流。
  • 享受无缝融合的管道模型与业界标准格式， OpenColorIO ，丰富的API和Python脚本支持
